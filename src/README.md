# Ultra Emoji Combat (Simulador de Lutas em Java)

Projeto desenvolvido para praticar **Programação Orientada a Objetos utilizando Java**.

O sistema simula lutas entre competidores em diferentes **categorias de peso**, organizadas em eventos.

---

## Funcionalidades

- Cadastro de lutadores
- Definição automática de categoria por peso
- Marcação de lutas entre lutadores da mesma categoria
- Simulação de resultados de luta (vitória, derrota ou empate)
- Organização de lutas em eventos
- Ranking de lutadores baseado em número de vitórias

---

## Conceitos de Programação Aplicados

Este projeto utiliza diversos conceitos importantes de Java:

- Programação Orientada a Objetos (POO)
- Encapsulamento
- Enum (Categorias de peso e resultado da luta)
- Interface
- Agregação entre classes
- Coleções (`List`)
- Streams e `Comparator` para ranking
- Organização em `packages`

## Como executar

Clone o repositório:

```bash
git clone https://github.com/JoaoPauloOlt/ultra-emoji-combat

Abra o projeto em uma IDE Java (IntelliJ, Eclipse ou VSCode).

Execute a classe: Main.java

O sistema irá simular um evento com lutas entre os lutadores cadastrados.
```
